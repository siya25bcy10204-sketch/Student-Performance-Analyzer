# Student-Performance-Analyzer
# Overview of the project
Student Performance Analyzer is a simple Python-based project that allows users to manage student records and automatically predict a student’s performance using predefined rule-based logic.
The system allows adding students, entering marks, viewing all student details, and searching for a particular student.
It is designed as an easy beginner-level project suitable for academic submissions, fundamentals in AI/ML, and Python practice.
No external libraries or files are required—this project runs entirely in Python IDLE.
# Features
 Add New Student
User can input student name and marks.
 Automatic Performance Prediction
Based on marks:
90–100 → Excellent
75–89 → Good
50–74 → Average
0–49 → Poor
 View All Students
Displays all saved student records with marks and predicted performance.
 Search Student
Find a student by name.
 Simple User Menu
Beginner-friendly menu for easy navigation.
 Single-File Project
Everything is inside one Python script—ideal for quick execution and academic submission.

# Steps to Install & Run the Project
1. Install Python
Download from the official website:
https://python.org
Make sure to check:
 Add Python to PATH
during installation.
2. Open Python IDLE
Go to:
Start Menu → Python → IDLE (Python GUI)
3. Create a New File
In IDLE:
File → New File
Paste the complete project code into it.
4. Save the File
Save as:
student_performance_analyzer.py
5. Run the Program
Press:F5
6. Use the Menu on Screen
Examples:
Press 1 to add student
Press 2 to view all students
Press 3 to search student
Press 4 to exit
